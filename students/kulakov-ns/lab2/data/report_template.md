# Лабораторная работа № 2

## Цель работы
Реализовать метод Random Forest, выполнить подбор гиперпараметров по OOB-оценке с использованием `GridSearchCV`, оценить важность признаков через OOB-перестановки и сравнить результат с эталонной реализацией `RandomForestClassifier` из `scikit-learn`.

## Выбранный датасет
Для работы использован датасет [Titanic](https://www.kaggle.com/competitions/titanic/overview).

Причины выбора:
- задача бинарной классификации;
- есть количественные признаки (`Age`, `SibSp`, `Parch`, `Fare`);
- есть категориальные признаки (`Pclass`, `Sex`, `Embarked`, `CabinDeck`);
- присутствуют пропуски, из-за чего требуется предварительная обработка.

## Подготовка данных
Из исходного датасета использовались признаки:
`Pclass`, `Sex`, `Embarked`, `CabinDeck`, `Age`, `SibSp`, `Parch`, `Fare`.

Признак `CabinDeck` был получен из поля `Cabin` как первая буква палубы.

Для подготовки данных были выполнены следующие шаги:
- категориальные признаки заполнены наиболее частым значением и закодированы через `OrdinalEncoder`;
- числовые признаки заполнены медианой;
- выборка разделена на обучающую и тестовую в отношении 80/20 со стратификацией.

## Описание метода
Собственная реализация `RandomForest` построена как ансамбль деревьев решений `DecisionTreeClassifier`.

Для каждого дерева:
- формируется bootstrap-выборка из обучающего набора;
- оставшиеся объекты используются как OOB-объекты;
- дерево обучается с ограничениями `{my_params}`.

Оптимальные параметры подбирались по OOB-оценке через `GridSearchCV`.
Итоговая OOB-оценка собственной модели: **{my_oob:.4f}**.
Итоговая OOB-оценка эталонной модели `sklearn`: **{sk_oob:.4f}**.

## Вывод
В ходе работы был реализован метод Random Forest с bootstrap-обучением деревьев, OOB-оценкой качества и OOB-перестановочной оценкой важности признаков. Подбор параметров через `GridSearchCV` по OOB позволил получить устойчивую модель без выделения отдельной валидационной выборки. Сравнение показало, что собственная реализация уступает библиотечной по времени обучения, но дает сопоставимое качество на тестовой выборке. Наиболее значимыми признаками для задачи выживания на Titanic оказались пол пассажира, тариф и возраст.
